Firestore Setup Instructions

1

Create Firebase Project

Go to Firebase Console and create a new project.

2

Enable Firestore

In Firebase Console → Build → Firestore Database → Create Database. Start in test mode.

3

Enable Authentication

In Firebase Console → Build → Authentication → Get Started → Enable Email/Password provider. Create your admin account.

4

Get Config

Go to Project Settings → General → Your apps → Add web app → Copy the config object and paste it in firebase-config.js.

5

Set Security Rules

In Firestore → Rules → Paste the following rules and publish:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /enquiries/{doc} {
      allow create: if true;
      allow read, update, delete: if request.auth != null;
    }
    match /devices/{doc} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
